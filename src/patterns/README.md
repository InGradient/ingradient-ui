# Patterns Layer

이 폴더는 `@ingradient/ui/patterns`의 구현을 가진다.

## Put Here

- app shell
- page header
- toolbar / filter bar
- sidebar shell
- split layout
- settings shell
- dashboard grid
- form section

## Do Not Put Here

- primitive-level input/display component
- product-specific workflow
- API-aware stateful screen

## Rule Of Thumb

- 화면 골격과 layout rhythm을 재사용하면 `patterns`
- business rule이나 product semantics가 들어가면 앱에 남긴다
