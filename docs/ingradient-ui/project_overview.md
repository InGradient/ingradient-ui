# ingradient-ui Project Overview

## 프로젝트 소개
`ingradient-ui`는 INGRADIENT 제품군이 공통으로 사용하는 디자인 시스템과 UI 패키지다.

## 목적
- 신규 프로젝트에서 제품 의미 없이 재사용 가능한 UI를 제공한다
- tokens, components, patterns, brand assets를 공통화한다

## 범위 / 비범위
- 범위: design tokens, primitives, components, patterns, brand, docs app
- 비범위: dataset export, training flow, camera workflow, permission meaning

## 주요 의존성
- build and publish pipeline
- consuming apps such as platform, medilabel, ai demos
