# Reference

이 폴더는 `@ingradient/ui`의 공유용 Markdown reference다.

목적은 두 가지다.

- interactive showcase를 열지 못하는 상황에서도 핵심 사용법을 빠르게 공유하기
- AI나 문서 도구에 그대로 붙여 넣을 수 있는 텍스트 기준 문서를 제공하기

## Relationship To Showcase

- showcase
  - live examples
  - states
  - related navigation
  - 시각 확인
- markdown reference
  - 빠른 공유
  - 검색과 복사
  - 텍스트 기반 요약

showcase가 사용자 문서의 기본 진입점이고, 이 폴더는 shareable snapshot이다.
